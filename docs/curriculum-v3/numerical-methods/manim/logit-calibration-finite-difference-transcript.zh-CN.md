# Logit 偏置校准：有限差分动画文字稿

## 0–8 秒：同一条残差曲线

数值方法进入有限差分。固定 12 个 logit，只增加一个标量偏置 $b$，定义

$$F(b)=\operatorname{mean}(\sigma(z+b))-0.62.$$

本节不训练模型，只问：只调用函数，能否检查 $b=0.35$ 附近的局部斜率？

## 8–19 秒：构造可重复残差

固定 logit 加上同一个 $b$，经过 sigmoid 得到 12 个概率；取平均后减去 0.62。在探针 $b_0=0.35$，固定输出为 $F(b_0)=-0.0607869881$，说明平均概率仍低于目标。

## 19–32 秒：割线逼近切线

前向差分使用 $F(b+h)$ 与 $F(b)$；中心差分使用 $F(b+h)$ 与 $F(b-h)$。中心差分同时读取两侧，主要截断项为 $O(h^2)$。几何上，两侧采样点的割线随窗口缩小而靠近 $b$ 处切线。

## 32–47 秒：完整步长扫描

Notebook 从 $h=10^{-1}$ 扫到 $10^{-12}$。前向差分在采样网格中的最佳点是 $h=10^{-7}$；中心差分的最佳点是 $h=10^{-5}$。中心差分绝对误差此时约 $2.339\times10^{-12}$。误差曲线先下降，再回升。

## 47–60 秒：两端误差来源

$h$ 过大时，割线窗口仍能看见曲率，Taylor 高阶项形成截断误差。$h$ 过小时，$F(b+h)$ 与 $F(b-h)$ 过于接近；相减损失有效数字，再除以 $2h$ 会放大浮点误差。稳定窗口位于二者之间，并随函数尺度、浮点精度和差分模板改变。

## 60–70 秒：与解析导数对齐

这个固定函数的解析导数是

$$F'(b)=\operatorname{mean}(p_i(1-p_i)).$$

在 $b=0.35$，解析结果为 0.1630982544。中心差分在 $h=10^{-5}$ 时得到 0.16309825440208314，绝对误差约 $2.339\times10^{-12}$。最小步长不等于最好步长。

## 70–78 秒：进入下一章

有限差分回答“这里有多陡”。下一章保持同一份数据和同一个 $F(b)$，转而求解 $F(b)=0$。匹配平均概率只解决一个标量方程，不等于完成完整概率校准。
