# Ames 条件数与敏感性：中文字幕稿

视频无配音。以下内容按最终 82 秒成片的固定分镜记录教学叙事。

## 0:00–0:08

小的方程残差不等于低敏感性。残差回答当前方程算得准不准；2-范数条件数回答输入稍变时解是否容易大幅变化。求解准确性和问题条件是两件事。

## 0:08–0:22

未缩放 Ames 设计矩阵的 2-范数条件数是 13044.220254；五个特征逐列减均值、除总体标准差后，条件数降为 3.222571。椭圆只表示不同方向的拉伸差异，是高维关系示意，不按实际尺度。标准化改善单位尺度，但不会增加信息，也不会消除真实共线性。

## 0:22–0:34

2-范数条件数为 κ₂(X)=σmax/σmin。若 X=UΣVᵀ 且满列秩，则 XᵀX=VΣ²Vᵀ，所以 κ₂(XᵀX)=κ₂(X)²。这里 3.222571²≈10.384962。形成正规方程会平方不同方向之间的尺度差异。

## 0:34–0:49

为了单独观察近共线性，诊断场景使用已经标准化的居住面积 z_living，以及由行号确定并已标准化的 η，再计算 z_duplicate=z_living+10⁻⁴η；相加后不再额外缩放。这一列只存在于诊断场景，不写回下载 CSV。X_near∈R²⁹²⁷ˣ⁷，仍然满列秩，却因两列近平行而病态，κ₂(X_near)=26644.503135。画面仍是高维关系示意，不按实际尺度。

## 0:49–1:02

沿最敏感方向加入相对目标扰动 10⁻⁵，得到相对系数变化 0.00329613；两者比值是本次观察到的 329.613418 倍放大。329.613418 不是条件数，只是这一次特定扰动与方向的观察值。实际近重复设计的条件数仍是 26644.503135。

## 1:02–1:16

再看透明的 2×2 场景。同一个 A=[[1,1],[1,1.0001]] 的条件数是 40002.000075。系统一使用 b=[2,2.0001]ᵀ，唯一解 x=[1,1]ᵀ；相邻系统使用 b′=[2,2.0002]ᵀ，唯一解 x′=[0,2]ᵀ。它们各自的残差都接近零。这是右端项从 b 变为 b′ 的两个相邻系统，不是同一个右端项凭空出现两个解。

## 1:16–1:22

可用处理包括缩放、删除或重参数化重复特征、收集更多有效信息，以及正则化。单纯提高浮点精度，不能补回近重复特征缺少的可辨识信息。下一章进入稀疏矩阵。
