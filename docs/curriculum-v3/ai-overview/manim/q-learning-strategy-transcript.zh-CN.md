# Q-learning 策略形成：中文字幕稿

视频无配音。以下文字按最终 90 秒成片中的时间段记录全部教学叙事。

## 0:00–0:08

这是一个 4×4 世界。起点在 (3,0)，目标在 (0,3)，障碍在 (1,1) 和 (2,2)。到达目标奖励 +10，普通一步奖励 −1，碰撞奖励 −3。固定随机种子是 7107。

## 0:08–0:18

state 是当前位置；action 是上、右、下、左；Q(s,a) 是“在状态 s 做动作 a，之后继续行动”的长期价值估计。Q value 不是成功次数，也不是即时 reward 本身。

## 0:18–0:34

新估计等于旧估计加学习率乘以“目标减旧估计”。Q-learning 更新式为 Q(s,a)←Q(s,a)+α[r+γ max Q(s′,a′)−Q(s,a)]。若旧值是 2，向右行动得到 +10，下一状态最大 Q 值是 4，则 target=10+0.9×4=13.6，correction=13.6−2=11.6，新值=2+0.5×11.6=7.8。reward 不是监督标签，Q value 也不等于即时 reward。

## 0:34–0:44

探索会偶尔尝试不同动作，帮助发现未知路线；利用会选择当前 Q value 最大的动作。训练中的 exploration rate 固定为 0.3。

## 0:44–1:10

使用 seed=7107、α=0.5、γ=0.9、exploration rate=0.3 分别训练到 episode 1、5、20 和 50。第 1 次训练的最后一集走 26 步、累计 reward −25；第 5 次为 17 步、−6；第 20 次为 6 步、+5；第 50 次为 12 步、−1。四次都到达目标。随机训练的最后一集 reward 不要求单调；随着 Q value 形成，贪心策略箭头逐渐明确。

## 1:10–1:22

评估时关闭探索，exploration rate=0。初始全零 Q 表在确定性平局规则下会反复撞向上边界，展示 8 步累计 reward −8。学习后的贪心路径依次上、上、上、右、右、右，6 步到达目标，累计 reward +5。

## 1:22–1:30

映射回学习助手：state 是当前掌握状态，action 是选择练习难度，reward 是稍后的学习改善，policy 是在每种掌握状态下选择下一步。策略仍需安全约束；奖励设计错误会鼓励意外行为。
