# 最低数学基础四章内容契约

**状态：** 已进入运行时实现

## 教学目标

这四章不是四篇互不相干的数学百科，而是一条从确定计算走向不确定性表达的连续路线：

1. 函数与映射：区分输入、参数、输出、目标与区间平均变化率。
2. 向量与矩阵：把一个样本写成向量，把同结构样本按行堆成矩阵，并读懂 shape 与轴语义。
3. 导数与局部变化：在固定批次上观察损失对单个参数的局部敏感度，并区分导数估计与参数更新。
4. 随机变量与概率分布：从确定 prediction 转向不确定结果，区分分布、样本和经验频率。

课程以详细讲解为主。函数与导数原有的三层练习不再进入运行时正文；每章只保留少量带原因和回看位置的 checkpoint。

## 共用数值主线

前三章固定同一组数值：

| 对象 | 数值 | 语义 |
|---|---:|---|
| 单样本特征 | `x = [2, 3]` | 两个有序特征 |
| 批次特征 | `X = [[2, 3], [1, 4]]` | axis 0 为样本，axis 1 为特征 |
| 权重 | `w = [4, -1]` | 与特征列一一对应 |
| 偏置 | `b = 5` | 每个样本共享的标量 |
| 目标 | `targets = [9, 7]` | 只在预测后用于计算误差 |
| 预测 | `predictions = [10, 5]` | `X @ w + b` |
| 残差 | `residuals = [1, -2]` | `predictions - targets` |
| MSE | `2.5` | 两个平方残差的平均 |
| 当前局部敏感度 | `dL/dw = [0, -5]`, `dL/db = -1` | 当前参数附近的一阶信息 |

概率章把任务切换为“下一局是否按时完成”：样本空间为 `{未按时, 按时}`，随机变量取值为 `{0, 1}`，示例分布为 `[0.30, 0.70]`。固定随机种子仅保证教学输出可复现，不改变概率含义。
