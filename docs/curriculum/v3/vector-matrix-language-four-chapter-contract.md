## 教学主线

这四章承接“最低数学基础”中的样本向量和批量矩阵，不重复解释标量、向量、矩阵的名称，而是回答四个更接近模型的问题：

1. 一个对象怎样进入有明确轴语义的特征空间？
2. 两个向量的“位置近”和“方向像”为什么会给出不同排序？
3. 一批向量怎样通过矩阵同时产生预测或新的表示？
4. 一个矩阵能表达哪些输出方向，又会完全忽略哪些输入变化？

## 共用数据

四章固定三个三维学习资源画像。三个轴依次表示代数、Python 和概率主题的相对活动强度：

| 名称 | 向量 | 用途 |
|---|---:|---|
| 查询画像 `q` | `[2, 1, 0]` | 当前学习需求 |
| 同方向画像 `same_direction` | `[4, 2, 0]` | 方向完全相同、长度更大 |
| 邻近画像 `nearby` | `[2, 1, 1]` | 欧氏距离更近、方向略有变化 |

批量矩阵按行堆叠为：

```text
profiles = [[2, 1, 0],
            [4, 2, 0],
            [2, 1, 1]]
```

固定数值结果：

- `distance(q, same_direction) = 2.236068`，`cosine = 1.0`。
- `distance(q, nearby) = 1.0`，`cosine = 0.912871`。
- `profiles @ [0.6, 0.3, 0.1] + 0.2 = [1.7, 3.2, 1.8]`。
- `W = [[1, 2, 0], [0, 0, 1]]` 的 rank 为 2。
- `null_direction = [-2, 1, 0]` 满足 `W @ null_direction = [0, 0]`。
- `q` 与 `q + null_direction = [0, 2, 0]` 不同，但经过 `W` 后都得到 `[4, 0]`。
