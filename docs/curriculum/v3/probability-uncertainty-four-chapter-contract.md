## 教学顺序

这条路线不把概率公式拆成互不相干的名词，而是连续回答四个问题：

1. 怎样从样本空间、随机变量和重复观察建立概率分布？
2. 怎样把期望或积分改写成可复现的随机样本平均，并理解估计误差？
3. 怎样把模型分数转成概率，再用似然、交叉熵、KL 和校准检查概率质量？
4. 怎样让一个概率分布通过转移矩阵逐步演化，并判断长期行为？

第一章保留现有零基础长课、分布实验和条件概率实验。后三章保留原 URL、正文、实验和理解回顾，通过 adapter 补充可复制代码、固定运行输出和章节交接。运行时新增 `probability-route`，不删除任何旧入口。

## 共用数值锚点

条件概率与模型概率使用同一个邮件信号案例：

```text
P(spam) = 0.08
P(signal | spam) = 0.82
P(signal | normal) = 0.12
P(signal) = 0.176
P(spam | signal) = 0.372727...
```

它说明命中率 `0.82` 不等于看到信号后的垃圾邮件概率；后验还要结合先验和误报率。似然章再把后验读成目标类别概率，说明真实类别概率越低，negative log likelihood 越高。

Monte Carlo 章使用固定种子 `7`，在单位正方形中抽样估计四分之一圆面积：

```text
N=100    pi_hat=3.08    abs_error=0.061593
N=1000   pi_hat=3.196   abs_error=0.054407
N=10000  pi_hat=3.1644  abs_error=0.022807
```

这些数值只用于复现一条样本路径，不宣称单次误差必然随样本数单调下降。典型误差尺度仍按 `1/sqrt(N)` 解释。

模型分布章固定 logits `[1.2, 0.3, -0.7]`，使用减去最大 logit 的稳定 softmax：

```text
probabilities = [0.642616, 0.261268, 0.096115]
target_nll = 0.442207
entropy = 0.859968
kl_p_q = 0.054711
```

马尔可夫章沿用天气列随机矩阵和雨天初始状态：

```text
day_1 = [0.2, 0.4, 0.4]
day_20 = [0.393443, 0.245902, 0.360656]
stationary_residual_l1 = 1.819e-09
```

## 每章完成标准

- `probability-route` 按上述四章顺序提供上一章、下一章和返回路线导航。
- 四章中英文标题、核心问题、变量名称和章节交接一致。
- 每章至少有一个可复制 Python/NumPy 代码块及对应运行输出。
- Monte Carlo 明确区分随机样本路径、估计器、偏差、方差和典型误差尺度。
- 概率损失章明确区分 posterior、likelihood、entropy、cross entropy、KL 与 calibration。
- 马尔可夫章检查概率非负、列和为 1、状态和为 1，并用稳定残差判断长期状态。
- 每个现有实验在每章只挂载一次，不新增大量练习或前端评分。
- 面向学习者的中文内容使用“已知信息、信号、观察、结果、记录”等普通表达，不使用内部验收词汇。
- Math Lab 首页提供该路线入口，旧模块 URL 和既有进度数据继续有效。

## 范围边界

- 不新增后端采样内核、账号系统、课程评分或大型练习库。
- 不把一条固定随机种子输出解释成统计保证。
- 不展开测度论、连续分布完整积分理论、MCMC 收敛证明或 Bayesian 深层推断。
- 不把 softmax 最大概率当成模型可靠性的充分条件。
- 不删除旧 URL、原实验、理解回顾或现有 localStorage 数据源。
