# V3.2 最小二乘与谱表征四章内容契约

**状态：** 已实施并验证
**范围：** `least-squares-fitting`、`eigenvalues-eigenvectors`、`svd`、`pca`

## 教学顺序

本阶段承接 rank 与 null space，形成下面的连续问题链：

1. 当目标不在 column space 中时，最小二乘怎样找到最近的可达输出？
2. 对同一批中心化数据，协方差矩阵有哪些只缩放、不转向的稳定方向？
3. SVD 怎样直接从数据矩阵拆出方向、尺度、rank 与最佳低秩近似？
4. PCA 怎样选择前几个方向，并用解释方差和重建误差说明保留与丢失？

现有四章长正文、误区反馈和交互实验继续保留。运行时移除页末长复习题，补入统一数据、可复制 NumPy 代码、固定运行输出和清晰的下一章交接。

## 共用数据

四个中心化的二维学习画像按行组成：

```text
X = [[-3, -1],
     [-1, -3],
     [ 1,  3],
     [ 3,  1]]
```

两列分别表示中心化后的代数活动和概率活动。每列均值为 0，因此这份矩阵可直接进入 covariance、SVD 与 PCA；若换成原始数据，必须先重新计算训练数据列均值。

最小二乘章固定目标：

```text
targets = [-7, -4, 5, 7]
```

固定数值结果：

- 最小二乘权重为 `[2.0625, 0.8125]`，预测为 `[-7.0, -4.5, 4.5, 7.0]`。
- 残差 `targets - predictions = [0.0, 0.5, 0.5, 0.0]`，SSE 为 `0.5`，并满足 `X.T @ residuals = [0.0, 0.0]`。
- 样本协方差矩阵为 `[[6.666667, 4.0], [4.0, 6.666667]]`。
- 协方差特征值为 `[10.666667, 2.666667]`，主方向为 `[0.707107, 0.707107]`。
- `X` 的奇异值为 `[5.656854, 2.828427]`，平方能量占比为 `[0.8, 0.2]`。
- rank-1 重建把四行映射为 `[-2,-2]、[-2,-2]、[2,2]、[2,2]`，Frobenius 误差为 `2.828427`，逐元素 RMSE 为 `1.0`。

## 每章完成标准

- 中英文解释、变量名称、shape、手算和 NumPy 输出保持一致。
- 代码至少检查二维 shape 和有限值，不静默接受转置、NaN 或错误轴。
- 最小二乘明确区分 residual、SSE 与参数，并解释正交条件。
- eigen 明确限定方阵，解释符号不唯一和 dominant eigendirection 的收敛条件。
- SVD 明确区分奇异值与其平方能量，不把任意小值自动判作噪声。
- PCA 明确中心化、解释方差、低维 scores 与重建的不同角色，不把 PCA 描述成监督分类器。
- 页面保留本地视觉资源和已有实验，不新增后端或浏览器 Python 内核。
